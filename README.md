# dotnet-aspnet-realworld-example-app
Exemplary real world app built with .NET Core + ASP.NET Core

## Getting started
```bash
cd Conduit
dotnet run
```

This project is generated by:
```bash
dotnet new gitignore
dotnet new webapi -o Conduit --no-https
```
